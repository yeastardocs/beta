Hang Up Call

This topic provides an overview of the Hang Up Call component, and describes its configuration as well as supported component connections.

Component introduction

Hang Up Call component is one of the end components to terminate a call flow. When callers are routed to the component, the call will be disconnected.

Component configuration

No configuration is needed for this component.

Component connections

Hang Up Call component comes with a built-in Call End branch that terminates the call, and the branch can be connected to only one Developer component.
