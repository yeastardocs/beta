Dial by Name

This topic provides an overview of the Dial by Name component, and describes its configuration as well as supported component connections.

Component introduction

Dial by Name component is one of the end components to terminate a call flow. It allows callers to reach extension user by entering the first three letters of the user's name.

Component configuration

No configuration is needed for this component.

However, to ensure that callers can successfully reach extension users, make sure the following settings are properly configured.

  1. Set Caller ID Name for extension.

    1. Go to Extension and Trunk > Extension, edit desired extension(s).
    2. In the User Information section, enter the user's first name and last name.
  2. Set display format for extension's Caller ID Name.

    1. Go to PBX Settings > Preferences.
    2. In the Name Display Format drop-down list, select a format to determine how extensions' Caller ID Names are displayed.

      When callers enter 3 letters, the system will search from the beginning of extensions' Caller ID Names based on the selected format.

Component connections

Dial by Name component comes with a built-in Call End branch that terminates the call, and the branch can be connected to only one Developer component.
